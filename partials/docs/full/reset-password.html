1. Включаем машину и переходим в меню grub с помощью нажатой клавиши "shift"
2. Редактируем загрузчик с помощью клавиши "e", а имеено параметры запуска ядра

где "rw" говорит, что диск будет смонтирован для чтения и записи, а "init=/bin/sh" то, что первым запустится bash shell
3. Запускаем с помощью комбинации клавиш "сtrl+x" и получаем root доступ к системе.
4. Сбрасываем пароль с помощью команды passwd
