Добавить нового пользователя 
sudo adduser newuser
Редактирование привилегий пользователей 
sudo /usr/sbin/visudo
Примеры использования утилиты dd 
dd if=kali.iso of=/dev/sdb bs=512k
sudo dd if=/path/to/iso/windows.iso of=/dev/sdg bs=4M; sync
Восстановление ntfs 
sudo ntfsfix  /dev/sdb1
Подключение к VPN по конфиг файлу 
sudo openvpn --config /path/to/config.ovpn
Создание soft ссылки(ярлыка) файла 
ln -s /webroot/home/httpd/test.com/index.php /home/vivek/index.php
Изменение владельца файла 
sudo chown -R $USER ~/.blabla

Мониторинг

Постоянный мониторинг логов 
tail -f file.log
Мониторинг выполнения команды при помощи утилиты watch 
watch -n 5 ls -agl

Работа с архивами

Создать архив 
zip -r myarch.zip mydir/*
Разархивироваие *.rar 
unrar e file.rar

AES Crypt

Шифрования(пароль - apples, файл - picture.jpg) 
aescrypt -e -p apples picture.jpg
Дешифровка(пароль - apples, файл - pictures.jpg.aes) 
aescrypt -d -p apples picture.jpg.aes

Cron

Редактирование задачи в кроне 
crontab -e
Стартовать скрипт каждые 10 минут 
*/10 * * * * /usr/bin/python ~/script_path/script.py

Find

Рекурсивный вывод содержимого файлов 
find [dir] -type f -exec cat \{} \;
Поиск файла или папки с помощью find 
find -name *file_name*
Использование chmod+find. 
find /opt/lampp/htdocs -type d -exec chmod 755 {} \;
Поиск файла по имени 
find -name "include-ext.js"
Отключить сессию гостя в ubuntu 
sudo gedit /usr/share/lightdm/lightdm.conf.d/50-ubuntu.conf
add "allow-guest=false"
