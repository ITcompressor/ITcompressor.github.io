Использование ssh 
ssh -p 22 steeldriver@192.168.1.102
Загружать и скачивать файлы через ssh 
scp your_username@remotehost.edu:foobar.txt /local/dir
Генерация ключей для работы через ssh 
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
Добавление публичного ключа на сервер 
cat ~/.ssh/id_rsa.pub | ssh root@your_server "cat >> /home/qwer/.ssh/authorized_keys"
Тунелирование ssh (т.е. проброс порта через подключение ssh) 
ssh -L 27017:localhost:27017 user@site.com
Генерация ключей для ssh 
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
Правильные права для ключей (set right permissions for ssh keys) 
chmod 600 .ssh/ -R
