Create zip archive 
zip -r myarch.zip mydir/*
Unzip rar archive 
unrar e file.rar
Unzip rar with keeping hierarchy 
unrar x file.rar
Create tar container 
tar -cvf name.tar /path/to/file1 /path/to/file2 /path/to/file3
Unzip tar container (archives) 
tar -xvf foo.tar
tar -xvzf foo.tar.gz
tar -xvjf foo.tar.bz2
Unzip tar.xz 
tar -xf file.tar.xz
Install 7z in debian like OS 
apt install p7zip
Examples for 7z 
7z x -so yourfile.tar.7z | tar xf - -C target_dir
7z x yourfile.7z
