Создать архив (create zip archive) 
zip -r myarch.zip mydir/*
Разархивироваие *.rar (unzip rar archive) 
unrar e file.rar
Разархивировать *.rar сохраняя иерархию папок (unzip rar keeping hierarchy) 
unrar x file.rar
Создать tar контейнер 
tar -cvf name.tar /path/to/file1 /path/to/file2 /path/to/file3
Распаковать tar контейнер (unzip tar container\archives) 
tar -xvf foo.tar
tar -xvzf foo.tar.gz
tar -xvjf foo.tar.bz2
Установка 7z в debian like OS (Install 7z in debian like OS) 
apt install p7zip
Примеры использования 7z (examples for 7z) 
7z x -so yourfile.tar.7z | tar xf - -C target_dir
7z x yourfile.7z
