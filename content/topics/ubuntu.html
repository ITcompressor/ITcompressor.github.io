Переконфигурация пакетного менеджера apt, если что-то пошло не так 
sudo dpkg --configure -a
Установка samba 
sudo apt install system-config-samba
Запуск samba 
sudo system-config-samba
Возможный фикс samba, если не находит нужного файла 
sudo touch /etc/libuser.conf
Установка bash-completion для автокомплита в консоли 
apt install bash-completion
Установка net-tools (ifconfig) 
apt install net-tools
Автологин -> вместо myusername подставляем имя нашего пользователя 
mkdir /etc/systemd/system/getty@tty1.service.d    
nano /etc/systemd/system/getty@tty1.service.d/override.conf

[Service]
ExecStart=
ExecStart=-/sbin/agetty --noissue --autologin myusername %I $TERM
Type=idle
Отключить сессию гостя в ubuntu 
sudo gedit /usr/share/lightdm/lightdm.conf.d/50-ubuntu.conf
add "allow-guest=false"
Манипуляции для включения rc.local (актуально для новых версий убунту 18+) 
# touch /etc/rc.local
# chmod +x /etc/rc.local
добавляем это 
#!/bin/sh -e
#
# rc.local
#
# This script is executed at the end of each multiuser runlevel.
# Make sure that the script will "exit 0" on success or any other
# value on error.
#
# In order to enable or disable this script just change the execution
# bits.
#
# By default this script does nothing.

exit 0
активируем 
sudo systemctl enable rc-local
Просмотр скрытых приложений в автозагрузке 
cd /etc/xdg/autostart
sudo sed --in-place 's/NoDisplay=true/NoDisplay=false/g' *.desktop
