Путь к исполняемым файлам 
/usr/local/bin
Просмотр скрытых приложений в автозагрузке 
cd /etc/xdg/autostart
sudo sed --in-place 's/NoDisplay=true/NoDisplay=false/g' *.desktop

Монтирование разделов

Посмотреть информацию о подключенных дисках 
sudo fstab -l
Редактирование автомонтирование разделов 
sudo nano /etc/fstab
Атомонтирование раздела ext4 
/dev/sda1 /home/qwer/DATA   ext4  defaults       0  0
Пример монтирования расшаренной виндой директорий 
nautilus smb://192.168.100.3/

Полезное

Просмотр запущенных процессов + фильтр по процессу 
ps aux | grep [command_name]
Информации о свободном месте на дисках 
df -h
Заполнение нулями свободную память на диске (в режиме восстановления) 
zerofree -v /dev/sdaX
