Монтирование раздела system для записи (необходимо для редактирования) 
mount -o rw,remount /system
Монтирование раздела system для чтения 
mount -o ro,remount /system
Путь к сохранёным паролям Wi-fi точек 
/data/misc/wifi/wpa_supplicant.txt
Файл hosts на андроиде 
/system/etc/hosts
Cкрипта в автозагрузке 
/system/etc/mkshrc
Блокировка ip через iptables (до перезагрузки устройства) 
iptables -A INPUT -s 65.55.44.100 -j DROP
Удаление блокировки 
iptables -D INPUT -s 65.55.44.100 -j DROP
Перепрошивка recovery (в режиме восстановления) 
fastboot flash recovery recovery.img
Разблокировка загрузчика (в режиме восстановления) 
fastboot oem unlock
Подключение устройства по adb 
adb connect 192.168.0.101:5555
Данная команда выведет список устройств, подключенных к ПК 
adb devices
Перезагрузка устройства с помощью adb 
adb reboot
Быстрая перезагрузка устройства в режим Recovery с помощью adb 
adb reboot recovery
Быстрая перезагрузка устройства в режим Bootloader с помощью adb 
adb reboot bootloader
Установка с компьютера пакета apk с помощью adb 
adb install <путь_к_apk>
Удаление пакета с помощью adb 
adb uninstall <package>
Загрузить файл на устройство с помощью adb 
adb push /path/to/local/file /mnt/sdcard/path/to/file
Скачать файл с устройства с помощью adb 
adb pull /mnt/sdcard/path/to/file /path/to/local/file
Просмотреть лог работы устройства с помощью adb 
adb logcat
После чего вы увидите динамический вывод системных сообщений Android. Если производить с устройством какие-либо манипуляции, это тут же будет отображаться через вывод соответствующих сообщений. Выход из режима отладки через комбинацию клавиш Ctrl+C. И при помощи ADB можно запустить на мобильном устройстве самый настоящий шелл Unix. 
adb shell
