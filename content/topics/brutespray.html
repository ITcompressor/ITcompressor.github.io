Install dependencies 
sudo apt install medusa nmap git
sudo pip install argcomplete
Get brutespray 
git clone git@github.com:x90skysn3k/brutespray.git
For quick access 
sudo cp brutespray/brutespray.py /usr/local/bin/brutespray
Collect info for brutespray (nmap --help) 
sudo nmap -sS -sV 192.168.1.0/24 -vv -oA test
Run bruteforce in 5 threads for 3 hosts (brutespray --help) 
brutespray -f test.nmap -t 5 -T 3
Output in (success login\pass) 
brutespray-output/
