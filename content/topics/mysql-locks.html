Просмотр текущих локов на БД (check locks) 
SELECT * FROM information_schema.innodb_locks il 
LEFT JOIN information_schema.innodb_trx itr ON itr.trx_id = il.lock_trx_id;
Еще один способ просмотра локов, но ифны дает меньше (check locks, but less info) 
SELECT * FROM information_schema.innodb_locks;
Просмотр транзакций (check transactions) 
SELECT * FROM information_schema.innodb_trx;
Просмотр всех процессов (check all processes) 
SHOW FULL PROCESSLIST;
Убить транзакцию (kill connection) 
kill connection 93309825;
Просмотр полного статуса бд (check full status) 
SHOW ENGINE INNODB STATUS;
