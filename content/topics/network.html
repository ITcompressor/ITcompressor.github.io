Использование утилиты nc

Запрос на 80 порт 
printf "HEAD / HTTP/1.0\r\n\r\n" | nc tut.by 80
Запрос с отправкой контента из файла 
nc localhost 1234 < test.txt

Использование утилиты curl

curl -X GET -H "Content-Type: [any_headers]" -d "{body=[any_params|body]}" https://itcompressor.github.io

Использование утилиты nmap

nmap -A -T4 ya.ru
ICMP пинг 
nmap -sn 192.168.1.0/24
TCP SYN Пинг 
nmap -sn -PS 192.168.1.0/24
TCP ACK Пинг 
nmap -sn -PA 192.168.1.0/24
UDP Пинг nmap -sn -PU 192.168.1.0/24 Пинг по IP Протоколу 
nmap -sn -PO 192.168.1.0/24
ARP Пинг 
nmap -sn -PR 192.168.1.0/24

Другие способы сканирования

Сканирование локальной сети на доступные машины 
echo 192.168.1.{1..254}|xargs -n1 -P0 ping -c1|grep "bytes from"
Путь пакета к ресурсу 
traceroute <any_ip_or_domain>

Использование wget

Скачать front-end (без rest запросов) 
wget -r -k -l 7 -p -E -nc https://itcompressor.github.io/
"Скачать и запустить" 
wget -q -O – https://fixubuntu.com/fixubuntu.sh | bash
Изменение mac-адреса в Linux с помощью утилиты ifconfig, где wlp3s0 название сетевого интерфейса 
sudo ifconfig wlp3s0 down
sudo ifconfig wlp3s0 hw ether 00:18:f3:4c:d6:6d
sudo ifconfig wlp3s0 up
