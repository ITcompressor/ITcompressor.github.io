Посмотреть активности в сети (check activity) 
netstat -an
Просмотр SSL у домена (check ssl) 
curl --insecure -v https://ya.ru 2>&1 | awk 'BEGIN { cert=0 } /^\* SSL connection/ { cert=1 } /^\*/ { if (cert) print }'
Запрос на 80 порт с помощью nc (send http request) 
printf "HEAD / HTTP/1.0\r\n\r\n" | nc tut.by 80
Запрос с отправкой контента из файла с помощью nc (send file by tcp) 
nc localhost 1234 < test.txt
Использование утилиты curl (curl examplse) 
curl -X GET -H "Content-Type: [any_headers]" -d "{body=[any_params|body]}" https://itcompressor.github.io
Использование утилиты nmap (А - палевный вариант) (Nmap scan if use -A - you will are detected) 
nmap -A -T4 ya.ru
ICMP пинг с помощью nmap (icmp ping) 
nmap -sn 192.168.1.0/24
TCP SYN Пинг с помощью nmap (tcp syn ping) 
nmap -sn -PS 192.168.1.0/24
TCP ACK Пинг с помощью nmap (tcp ack ping) 
nmap -sn -PA 192.168.1.0/24
UDP Пинг с помощью nmap (udp ping) 
nmap -sn -PU 192.168.1.0/24
Пинг по IP Протоколу с помощью nmap 
nmap -sn -PO 192.168.1.0/24
ARP Пинг с помощью nmap (arp ping) 
nmap -sn -PR 192.168.1.0/24
Скан по портам с помощью nmap (scan ports) 
nmap localhost -p 34000-36000
nmap localhost -p 8080
Сканирование локальной сети на доступные машины (scan network) 
echo 192.168.1.{1..254}|xargs -n1 -P0 ping -c1|grep "bytes from"
Путь пакета к ресурсу (traceroute for resource) 
traceroute <any_ip_or_domain>
Сканирование всех доменов по сертификату определенного домена (see all certs) 
true | openssl s_client -showcerts -connect habrahabr.ru:443 2>&1 |openssl x509 -text | grep -o 'DNS:[^,]*' | cut -f2 -d:
Скачать front-end (без rest запросов) c помощью wget (download front-end) 
wget -r -k -l 7 -p -E -nc https://itcompressor.github.io/
"Скачать и запустить" c помощью wget (download and run) 
wget -q -O – https://fixubuntu.com/fixubuntu.sh | bash
Изменение mac-адреса в Linux с помощью утилиты ifconfig, где wlp3s0 название сетевого интерфейса (change mac address) 
sudo ifconfig wlp3s0 down
sudo ifconfig wlp3s0 hw ether 00:18:f3:4c:d6:6d
sudo ifconfig wlp3s0 up
