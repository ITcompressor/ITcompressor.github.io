Start network network-manager 
sudo service network-manager start
Connect to wifi from terminal 
nmcli device wifi connect 'wifi_name_here' password 'wifi_password_here'
Check network activity 
netstat -an
Check SSL for domain 
curl --insecure -v https://ya.ru 2>&1 | awk 'BEGIN { cert=0 } /^\* SSL connection/ { cert=1 } /^\*/ { if (cert) print }'
Send http(tcp) request to 80 port by nc 
printf "HEAD / HTTP/1.0\r\n\r\n" | nc tut.by 80
Send tcp request from file by nc 
nc localhost 1234 < test.txt
To instance curl GET request 
curl -X GET -H "Content-Type: [any_headers]" -d "{body=[any_params|body]}" https://itcompressor.github.io
