Найстройки баша хранятся в (Shell settings) 
.bashrc - linux like
.bash_profile - macos
Установка bash-completion для автокомплита в терминале (autocomplete in a terminal) 
apt install bash-completion
Добавить папку с исполняемыми файлами в баш для быстрого доступа (add path for execute files) 
export PATH=$PATH:~/s/
Красивый терминал MacOS (pretty terminal for MacOS) 
export PS1="\[\033[36m\]\u\[\033[m\]@\[\033[32m\]\h:\[\033[33;1m\]\w\[\033[m\]\$ "
export CLICOLOR=1
export LSCOLORS=ExFxBxDxCxegedabagacad
alias ls='ls -GFh'
Pretty terminal for android (kali linux theme) 
export PS1="\[\033[31m\]\h:\[\033[33;1m\]\w\[\033[m\]\$ "
export CLICOLOR=1
export LSCOLORS=ExFxBxDxCxegedabagacad
alias ls='ls -GFh'
Если не работает автокомплит (if bash-completion not working) 
if ! shopt -oq posix; then
  if [ -f /usr/share/bash-completion/bash_completion ]; then
    . /usr/share/bash-completion/bash_completion
  elif [ -f /etc/bash_completion ]; then
    . /etc/bash_completion
  fi
fi
