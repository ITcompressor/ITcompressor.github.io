Запуск сервера приложений (домена) в режиме дебага (Start web-server with debug) 
as-install/bin/asadmin start-domain --debug domain1
Остановка сервера приложений (домена) (Stop web-server) 
as-install/bin/asadmin stop-domain
Деплой .war файла (Depoloy .war file) 
as-install/bin/asadmin deploy war-name
Список задеплоиных приложений (List deployed applications) 
as-install/bin/asadmin list-applications
Раздеплоить .war файл (Undeploy .war file) 
as-install/bin/asadmin undeploy war-name
Установка пароля на админку (set admin's panel password) 
as-install/bin/asadmin change-admin-password
Включить доступность админки для внешних подключений (enable admin panel for external connections) 
as-install/bin/asadmin enable-secure-admin
Расположение админки (Admin panel) 
http://localhost:4848
Путь к автодеплю (autodeploy path) 
as-install/domains/domain1/autodeploy
Путь к настройке переменных среды, там можно установить путь к джаве (env config path, example for JVM_HOME) 
as-install/config/asenv.conf
