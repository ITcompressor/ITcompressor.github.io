Перезагрузка 
shutdown -r
Выключение 
shutdown -s
Завершение сессии 
shutdown -l
Выполнение по времени, где xx количество секунд для выключения\перезагрузки\завершения сессии 
shutdown -t xx
Запуск диалогового окна для завершения работы 
shutdown -i
или 
shutdown -a
Дополнительная опция для принудительного выполнения 
-f
Изменить пароль 
net user current_user password
Добавить нового пользователя 
net user /add new_user password
Включение и выключение учетной записи администратора 
net user Администратор /active:yes
net user Администратор /active:no
С помощью таких манипуляций можно вызвать консоль через специальные возможности 
move d:\windows\system32\utilman.exe d:\windows\system32\utilman2.exe
copy d:\windows\system32\cmd.exe d:\windows\system32\utilman.exe
