Дамп процесса java с помощью утилиты jmap (dump java process) 
jmap -dump:format=b,file=cheap.bin pid_value
Вывод стека работающего java процесса (view java process stack) 
ps aux | grep "java"
jstack -l [java_pid] > output.txt
Просмотр байткода (view bytecode) 
javap -verbose My.class
Установка кастомных либ (maven install custom libs) 
mvn install:install-file -Dfile=[path-to-file] -DgroupId=[group-id] -DartifactId=[artifact-id] -Dversion=[version] -Dpackaging=jar
mvn install:install-file -Dfile=[path-to-file] -DpomFile=[path-to-pomfile]
mvn install:install-file -Dfile=[path-to-file]
Пример установки библиотеки для maven (example for install lib from maven) 
mvn install:install-file -Dfile=/path_to_jar/sqljdbc4.jar -DgroupId=com.microsoft.sqlserver -DartifactId=sqljdbc4 -Dversion=4.0 -Dpackaging=jar
Запуск спринг бутового приложения с переопределенными пропертями (run spring boot app with overrided properties) 
java -jar conbot-idf.jar --spring.config.location=telegram-bot.properties,idf.properties
