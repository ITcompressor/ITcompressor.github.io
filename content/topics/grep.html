Получение списка пользователей имеющих права суперпользователя 
grep -Po '^sudo.+:\K.*$' /etc/group
Показаная ниже команда производит поиск подстроки 12.00 в файле /home/david/backup/log.txt и отображает все строки где эта подстрока встречается. 
$ grep '12:00' /home/david/backup/log.txt
Команда (с использованием ключа -v) наоборот покажет только те строки где подстрока ’12:00′ не встречается. 
$ grep -v '12.00' /home/david/backup/log.txt
Команда поиска файлов оканчивающиеся на .c с текстом 'delay' 
$ grep -l 'delay' /code/*.c
Эта команда уже более сложная и состоит из комбинации двух команд grep. Первая ищет строки которые начинаются со слова ‘bay’ а вторая строки которые заканчиваются на слово ‘watch’. 
$ grep -w '\<bay' * $ grep -w 'watch\>' *
Покажет только те файлы у которых установлены нужные вам права доступа. 
$ ls -l | grep rwxrwxrwx
Вывод постранично список mp3 файлов найденых в текущей директории 
$ du | grep 'mp3' | more
Команда отобразит все строки в файле /home/david/script1 которые начинаются с символа ‘#’. Определение тип ‘^#’ означает что символ ‘#’ должен быть первым символом с троке. 
$ grep '^#' /home/david/script1 | more
Поиск строки содержащие в первом символе цифры от 0 до 9 а потом выводит только те строки которые не попали в результаты поиска. 
$ grep -v '^[0-9]' /home/david/backup/log.txt | more
Фильтр с помощью grep для ls 
ls -agl | egrep "text"
Рекурсивный поиск куска текста в директориях и файлах по регулярке 
grep -r "6419525" 2019-01-**
Пример грепа с последующим выводом лога от необходимого места 
grep -n "text" file.log
